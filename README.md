# bc-16-Quiz_App
This contains files for the project "Quiz_App"

The aim of this project it to create a Quiz app that:
	a) Contains the lists of all the available quizzes in my library.
	b) Allows the user to import quizzes from the JSON files.
	c) Shows the user the list of questions in every quiz as well as different answer options.
	d) Allows the user to choose the preffered answers from the choices provided.
	e) Shows the user results depending on their choices after each question and the accumulated results.

Additional details:
	a) The app should be able to identify "Invalid Inputs" that are not part of the choices given.
	b) The app will have a countdown timer.
